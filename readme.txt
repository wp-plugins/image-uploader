This plugin will allow client side users to upload images to a site.

== Description ==

This plugin will allow users to upload images to the site, but they will be restricted to an exact size depending on the type of upload (graphics, websites, etc)

It can be easily added to a page or post by using the code [imageuploaderform].

A link to the file and the submission information entered will then be emailed to the specified person in the admin configuration settings.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `image-uploader` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the settings as required
4. Add the submission form in a post or page by using the code [imageuploaderform]
